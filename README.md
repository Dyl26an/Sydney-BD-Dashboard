# Sydney Growth Intelligence V3

A Streamlit app for encrypted Excel merchant reports.

## Files
- app.py
- requirements.txt
- README.md

## Deploy
Upload these three files to GitHub and deploy via Streamlit Cloud.

## Features
- Password-protected Excel upload
- BD name display
- KPI dictionary
- BD ranking
- Compare Me vs Top BD
- Merchant Intelligence with merchant names
- Area Intelligence
- Opportunity Finder
- Learn From Best
